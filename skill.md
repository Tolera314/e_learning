# Next Features to Build (Expert Roadmap)

## 1. Assignment Management System

Students submit assignments and instructors review them.

### Features

* Upload assignment files
* Deadline management
* Instructor grading
* Feedback system
* Resubmission support

### Prompt

```
Build an Assignment Management module for an LMS platform.

Students must be able to:
• View assignments for each course
• Upload assignment submissions (PDF, DOCX, ZIP)
• See deadlines and submission status
• View grades and instructor feedback

Instructors must be able to:
• Create assignments
• Set deadlines and instructions
• View all submissions
• Grade assignments
• Provide written feedback
• Allow resubmissions

Admin must be able to:
• Monitor submission activity
• Track grading progress
• Generate assignment reports

System requirements:
• Store submissions securely
• Validate file types and size
• Track timestamps
• Send deadline reminders
• Maintain audit logs
```

# 2. Discussion Forum System

### Features

* Threaded discussions
* Replies
* Instructor pinned posts
* Moderation

### Prompt

```
Build a Course Discussion Forum system for the LMS.

Students must be able to:
• Create discussion posts
• Reply to other posts
• Like or react to posts
• Search discussions
• Follow threads

Instructors must be able to:
• Create announcement posts
• Pin important discussions
• Moderate discussions
• Delete inappropriate content

Admins must be able to:
• Monitor discussion activity
• Flag inappropriate posts
• Review moderation logs

System requirements:
• Threaded comments
• Real-time updates

• Performance optimized queries

# 3. Notification & Messaging System

Keeps users informed about activities.

### Features

* Course notifications
* Quiz reminders
* Messaging between users

### Prompt

```
Build a Notification and Messaging System for the LMS.

Students must receive notifications for:
• New course content

• Assignment deadlines
• Live class reminders
• Instructor announcements

Students and instructors must be able to:
• Send direct messages
• View conversation history
• Receive real-time notifications

Admins must be able to:
• Broadcast platform announcements
• Monitor messaging activity
• Manage notification templates

System performance requirements:

• Message storage optimization
• Notification queue processing
• High scalability for large user base
